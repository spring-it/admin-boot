# 🎉admin-boot

### 介绍
这是一个基础的企业级基础后端脚手架项目，主要由springboot为基础搭建，后期整合一些基础插件例如：redis、xxl-job、flowable、minioio、easyexcel、skyWalking、rabbitmq。

项目地址：<https://gitee.com/springzb/admin-boot>
